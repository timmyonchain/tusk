# 🐘 TUSK

**The Feedback Layer for Modern Teams**

TUSK is a premium form builder and feedback platform. Create beautiful forms, collect structured responses, and manage everything from a clean admin dashboard.

Live at [tusk.ink](https://www.tusk.ink)

---

## Features

- **Form Builder** — Drag and drop fields, 8 input types, 8 ready-made templates
- **Shareable Links** — Every form gets a unique link and QR code
- **Password Protection** — Make forms private with a password
- **Submission Limits** — Set a max number of responses per form
- **Revoke Links** — Close a form instantly at any time
- **Receipt IDs** — Every respondent gets a unique submission receipt
- **Admin Dashboard** — Filter, search, update status, and export CSV
- **Real Auth** — Email signup and login powered by Supabase
- **Mobile First** — Fully responsive, works beautifully on any device

---

## Tech Stack

- **Frontend** — React + Vite
- **Styling** — Tailwind CSS + custom design system
- **Auth + Database** — Supabase
- **Animations** — Framer Motion
- **Icons** — Lucide React
- **Hosting** — Vercel
- **Domain** — tusk.ink

## Getting Started

```bash
git clone https://github.com/timmyonchain/tusk.git
cd tusk
npm install
```

Create a `.env` file in the root:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Then run:

```bash
npm run dev
```
